# Bright-Coffee-Shop
☕ Overview
This project explores sales, customer behavior, and operational performance for Bright Coffee Shop, using real‑world data to uncover insights that support better decision‑making. The analysis covers data cleaning, exploratory analysis, visualization, and actionable recommendations for the business.


Objectives


Understand sales trends across products, days, and store locations

Identify top‑performing and underperforming product categories

Analyze customer purchasing patterns

Provide data‑driven recommendations to improve revenue and efficiency

🛠️ Tools & Technologies

Miro

Canva

SQL / Databricks

Microsoft Excel

TASK 1: PLANNING AND ARCHITECTURE

Developed a data flow and ETL architecture using Miro

Developed a project planning Gantt chart to show the process of completing the project


TASK 2: DATA PROCESSING IN DATABRICKS

Transformed raw Excel data to csv

Loaded to Databricks and performed key cleaning and transformation

Standardized price formatting

Created transaction-time_bucket (e.g 3 hour intervals)

Calculated total_revenue per transaction


TASK 3: DATA ANALYSIS IN EXCEL

Created pivot tables and dashboard showing

Revenue by product category, day name, month name and product detail

Peak sales intervals

Best- and worst- selling items


TASK 4: Executive presentation delivering key takeaways backed by visuals

📝 Recommendations
Introduce morning combo deals to increase average order value

Promote cold beverages during warmer months

Revisit pricing for low‑margin items

Use loyalty data to personalize offers
